# 🀄 孔明棋（Peg Solitaire）

> 基于 C++ 实现的孔明棋游戏，支持多种棋盘布局与人机对战。

---

## 📖 项目简介

**孔明棋**（又称独立钻石棋、Peg Solitaire）是一种单人智力棋盘游戏。棋盘上初始摆放若干棋子，玩家通过「跳吃」操作，让一枚棋子跳过相邻棋子并将其移除，目标是最终只剩下一枚棋子（或达到指定目标状态）。

本项目为课程期末大作业，使用 **C++** 实现核心游戏逻辑，并通过图形界面进行交互。

---

## 🚀 功能特性

- ♟️ 经典孔明棋棋盘（英式十字形布局）
- 🎮 交互式操作：点击选择棋子并移动
- 🤖 提供自动求解 / 提示功能
- 🖼️ 图形化棋盘界面（PNG 素材渲染）
- 📊 步数统计与游戏结果判断

## 🛠️ 构建与运行

### 环境要求

| 工具 | 版本要求 |
|------|---------|
| C++ 编译器 | C++17 或更高 |
| Visual Studio | 2019 / 2022（Windows） |
| g++ / clang++ | macOS / Linux |

### Windows（Visual Studio）

1. 用 Visual Studio 打开 `孔明棋作业.sln`
2. 选择 **Debug** 或 **Release** 配置
3. 按 `Ctrl+Shift+B` 编译，`F5` 运行

### macOS / Linux（命令行）

```bash
cd 孔明棋作业
g++ -std=c++17 -o main main.cpp
./main
```

---

## 🎮 游戏规则

1. 棋盘初始状态：中心空位，其余全部摆放棋子
2. 每次操作：选择一枚棋子，跳过相邻棋子，落入对面的空位，被跳过的棋子移除
3. 只能水平或垂直方向跳跃，不可斜跳
4. 当没有合法移动时游戏结束
5. **胜利条件**：棋盘上仅剩 1 枚棋子，且位于中心位置
